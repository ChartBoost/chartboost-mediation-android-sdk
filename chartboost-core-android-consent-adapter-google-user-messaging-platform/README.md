# Chartboost Core Consent Google User Messaging Platform Adapter

The Chartboost Core Google UMP adapter mediates Google User Messaging Platform SDK via the Chartboost Core SDK.

## Minimum Requirements

| Plugin              | Version |
|---------------------|---------|
| Chartboost Core SDK | 1.0.0+  |
| Android API         | 21+     |

## Integration

In your `build.gradle`, add the following entry:
```
    implementation "com.chartboost:chartboost-core-adapter-google-ump:1.3.0.0.0"
```
